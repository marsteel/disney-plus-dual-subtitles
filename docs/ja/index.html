Disney+ 二ヶ国語字幕

Disney+ で究極の二ヶ国語字幕体験を提供する、高性能な Chrome & Edge 拡張機能です。

✨ 主な機能

♾️ 字幕の安定性

  • 「スライディングウィンドウ」を自動処理
  • 200秒の制限を超えて継続再生が可能
  • 絶対 VTT アンカーによる正確なタイミング

🎯 圧倒的な視覚体験

  • 揺るぎない 2 スロットレイアウト
  • 字幕の「ジャンプ」や揺れを完全に排除
  • ネイティブ全画面表示をフルサポート

⚡ 高精度な同期

  • Sticky Offset が UI 時計のズレを解消
  • Visual Snap による自動修復機能
  • Shadow DOM からのミリ秒単位の抽出

🎨 自由なカスタマイズ

  • 色、サイズ、太さ、背景をリアルタイム調整
  • Hot Reload により保存後すぐに反映
  • 7 つの言語に完全対応した UI
