Politique de Confidentialité

Disney+ Double Sous-titres est une extension Chrome & Edge engagée dans la protection de votre vie privée.

Dernière mise à jour : 13 mars 2026

✨ Aperçu

Disney+ Double Sous-titres est conçu pour offrir une expérience de double sous-titrage premium. Nous ne collectons, ne stockons, ni ne transmettons AUCUNE donnée personnelle à nos serveurs. Nous n'avons pas de serveurs.

📦 Ce que nous stockons localement

Les données suivantes sont stockées uniquement localement dans votre navigateur via chrome.storage.sync :

  • **Paramètres de langue** : Langues primaires et secondaires sélectionnées.
  • **Préférences de style** : Paramètres de couleur, taille, arrière-plan et poids de la police.
  • **État de l'extension** : État de l'interrupteur marche/arrêt.

Tous les paramètres sont stockés localement et synchronisés sur vos appareils via la synchronisation du navigateur (si activée). Nous n'avons pas accès à ces données.

📡 Services Tiers

Lorsque vous utilisez Disney+ Double Sous-titres :

  • Les fragments de sous-titres sont téléchargés directement depuis les serveurs Disney+ vers votre navigateur.
  • Nous n'interceptons, ne stockons, ni ne visualisons vos données de visionnage ou le contenu des sous-titres.
  • Aucune donnée n'est envoyée à des services tiers d'IA ou de traduction.
🔧 Explications sur les Permissions

storage

Pour enregistrer vos préférences de langue et vos styles de sous-titres personnalisés localement dans votre navigateur.

host_permissions

Pour accéder à *.disneyplus.com afin d'insérer les doubles sous-titres et récupérer les segments nécessaires.

🚀 Sécurité des données & Open Source

Tous les paramètres sont stockés à l'aide du stockage intégré du navigateur. Comme Disney+ Double Sous-titres est open source, vous pouvez consulter le code source complet sur GitHub pour vérifier nos affirmations de confidentialité à tout moment.
