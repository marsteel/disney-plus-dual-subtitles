隱私政策

Disney+ 雙語字幕是一款 Chrome & Edge 瀏覽器擴充功能，我們高度重視您的隱私。

最後更新：2026年3月13日

✨ 概述

Disney+ 雙語字幕旨在提供極致的雙語觀影體驗。我們**不會**收集、存儲或向我們的服務器傳輸任何個人數據。我們沒有任何服務器。

📦 本地存儲說明

以下數據僅通過 chrome.storage.sync 存儲在您的瀏覽器本地：

  • **語言設置**：選定 的主語言和副語言。
  • **樣式偏好**：字體顏色、大小、背景色和粗細設置。
  • **擴充功能狀態**：開啟/關閉開關狀態。

所有設置均存儲在瀏覽器本地，並可通過瀏覽器同步功能（如已開啟）在您的設備間同步。我們無法訪問這些數據。

📡 第三方服務

當您使用 Disney+ 雙語字幕時：

  • 字幕片段直接從 Disney+ 服務器下載到您的瀏覽器。
  • 我們不會攔截、存儲或訪問您的觀看數據或字幕內容。
  • 不會將任何數據發送到第三方 AI 或翻譯服務。
🔧 權限說明

storage

用於在瀏覽器本地保存您的語言偏好和自定義字幕樣式。

host_permissions

用於訪問 *.disneyplus.com 以注入雙語字幕並獲取必要的字幕切片。

🚀 數據安全與開源

所有設置均使用瀏覽器內置存儲。由於本项目完全開源，您可以隨時在 GitHub 上檢查源代碼以驗證我們的隱私聲明。
