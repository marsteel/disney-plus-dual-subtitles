Disney+ Dual Subtitles

A high-performance Chrome extension for a premium dual-subtitle experience. Designed for cinematic stability and perfect synchronization.

🏆 Why Choose Us?

♾️ Infinite Streaming

Never lose subtitles during long movies. Our engine automatically handles Disney's sliding window manifests, ensuring continuous playback beyond the 200s cutoff.

🎯 Zero-Jitter Display

A stable, dual-slot layout prevents subtitles from shaking or jumping when line counts change. Clean, professional, and cinematic.

⚡ High-Precision Sync

Triple-tier synchronization engine (Shadow DOM + Sticky Offset + Visual Snap) keeps subtitles frame-perfectly aligned with the video.

🌍 Professional i18n

Full localization for 7 languages including English, Chinese, Japanese, and more. Precise language variant detection (e.g., zh-hk, zh-hans).

🎨 Advanced Styling

Complete control over font size, colors, background opacity, and weight. Changes apply instantly without page reloads.

🛠️ Hot Reloading

Tweak your settings in the popup and see them live immediately. No more refreshing the page every time you change a color.

🛠 Advanced Architecture

Absolute VTT Anchoring

We solve Disney's manifest quirks by parsing X-TIMESTAMP-MAP in VTT headers. Every segment is anchored to the movie's global timeline, ignoring relative offsets that cause sync drift in other extensions.

X-TIMESTAMP-MAP=MPEGTS:1791000,LOCAL:00:00:00.000

🚀 Installation

  1. Download the latest release from GitHub.
  2. Open Chrome and navigate to chrome://extensions/.
  3. Enable Developer Mode.
  4. Click Load unpacked and select the project folder.
  5. Open Disney+ and start your dual-language journey!
