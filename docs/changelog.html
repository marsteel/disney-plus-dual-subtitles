Changelog

Version history and release notes for Disney+ Dual Subtitles extension.

⚠️ Important Notice

Due to unpredictable changes in the video platform's architecture, the extension may occasionally stop working. If you encounter issues, please try reloading the webpage. If subtitles fail to appear, please try reloading the entire browser page.

The extension operates with minimal permissions. Future versions may request additional permissions as needed. If the extension is disabled due to a permission update, please manually re-enable it; this is a standard browser security procedure and does not mean the extension is harmful.

The author maintains this project on a best-effort basis. Your feedback is valuable—please report any issues on the GitHub Issues page.

v1.3.0 March 15, 2026

🎯 Accuracy & Stability Fixes

  • Timeline Sync (Anti-Drift): Implemented auto-detection for absolute vs. relative VTT files, fixing a major 7.5-minute subtitle drift issue.
  • Optimized Preloading: Rewrote the download engine to prioritize segments nearest to the playhead, ensuring instant subtitles after seeking or language switching.
  • Extension Context Safeguards: Prevented "Extension context invalidated" errors by adding validity checks to all browser API calls.
  • Robust VTT Parsing: Added explicit support for skipping non-cue blocks like STYLE and NOTE in subtitle assets.
  • Host Permissions: Added support for Disney's subtitle delivery domains (dssott.com) to ensure unblocked access.
  • Smart Changelog Popup: The changelog now only appears when the extension version actually changes, avoiding popups during local reloads or simple updates.
v1.2.1 March 14, 2026

🐞 Bug Fixes & Stability

  • Subtitle Persistence Fix: Resolved a critical bug where subtitles would remain on screen after navigating away from the video player to other pages.
v1.2.0 March 13, 2026

🚀 Smart Experience & Notifications

  • Auto-Changelog: The extension now automatically opens this page upon installation or system updates to keep you informed.
  • Smart Localization: Automatically detects your browser's language during first setup to set the best default subtitle language.
  • Loading Stability: Fixed Google Translate widget loading for local file viewing.
v1.1.0 March 13, 2026

🎭 Brand Identity & UI Enhancements

  • Repository Maintenance: Added .gitignore and cleaned up temporary system files.

🎉 Initial Release
