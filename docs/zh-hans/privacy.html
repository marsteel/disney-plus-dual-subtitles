隐私政策

Disney+ 双语字幕是一款 Chrome & Edge 浏览器扩展，我们高度重视您的隐私。

最后更新：2026年3月13日

✨ 概述

Disney+ 双语字幕旨在提供极致的双语观影体验。我们**不会**收集、存储或向我们的服务器传输任何个人数据。我们没有任何服务器。

📦 本地存储说明

以下数据仅通过 chrome.storage.sync 存储在您的浏览器本地：

  • **语言设置**：选定的主语言和副语言。
  • **样式偏好**：字体颜色、大小、背景色和粗细设置。
  • **扩展状态**：开启/关闭开关状态。

所有设置均存储在浏览器本地，并可通过浏览器同步功能（如已开启）在您的设备间同步。我们无法访问这些数据。

📡 第三方服务

当您使用 Disney+ 双语字幕时：

  • 字幕片段直接从 Disney+ 服务器下载到您的浏览器。
  • 我们不会拦截、存储或访问您的观看数据或字幕内容。
  • 不会将任何数据发送到第三方 AI 或翻译服务。
🔧 权限说明

storage

用于在浏览器本地保存您的语言偏好和自定义字幕样式。

host_permissions

用于访问 *.disneyplus.com 以注入双语字幕并获取必要的字幕切片。

🚀 数据安全与开源

所有设置均使用浏览器内置存储。由于本项目完全开源，您可以随时在 GitHub 上检查源代码以验证我们的隐私声明。
